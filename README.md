# Traffic Analyzer

A beginner Python project built as part of my MSc 
Transportation Engineering preparation (May–July 2026).

## What it does
- Loads a small traffic dataset from a CSV file
- Computes average speed across all time slots
- Identifies the peak hour by vehicle count
- Classifies each time slot as Congested or Free Flow

## Stack
- Python 3.13
- pandas 2.3.1

## How to run
1. Clone the repository
2. Make sure pandas is installed: `pip install pandas`
3. Run: `python main.py`

## What I learned
- How to use pandas in a real project rather than in isolation
- Why apply() is needed for row-by-row column operations
- The difference between the Python shell and the terminal
- Variable naming discipline and output mindset
